# Yapılacaklar Listesi

## Öncelikli Görevler
- bu proje, JSON formatında verileri, dart nesnelerine dönüştürmeye yarayacak
- bir metin alanı olacak, bu alan oldukça büyük olacak çünkü buraya büyük JSON verileri yapıştırılacak
- bu yapıştırılan veriler, dart nesnelerine(class'larına) dönüştürülecek
- bu dart nesnesine/class'ına isim verilebilecek(textfield yardımıyla)
- oluşturulan dart classında değişkenler;
final, nullable(optional), olabilecek, ancak bunlar switch yardımıyla seçilecek(optional), ancak default seçili gelecek.
- dart class'ında fromJson, toJson metodları olacak(optional switch yardımıyla, default seçili gelecek)
- dart class'ında copyWith metodu oluşturulabilecek(optional switch yardımıyla)
- dart class'ında, daha doğrusu constructor'da değişkenler required olarak işaretlenebilecek(optional switch yardımıyla)
- dart class'ında sadece tipler(değişkenler) oluşturulabilecek, herhangi bir ekstra method veya function içermeyecek(optional switch yardımıyla)
- dart class'ı, data class haline getirebilecek, yani == ve hashcode metodları override edilip oluşturulan class, reference type'dan value type'a dönüştürülebilecek(optional switch yardımıyla)
- oluşturulan dart nesnesini kopyalayabileceğimiz bir buton olacak
